The functionality of the monitor accessible by the user is through the user interface.  The user interface is made up of a simple command interpreter, command line editor and command table.  This section categorizes each of the commands, provides a brief description of the command, and points to a more detailed page describing the complete usage of the command in the monitor.

Memory Display and Modification Commands

cast
    overlay a structure definition onto a block of  memory for display
cm
    copy memory
dm
    display memory
flash
    flash memory operations
fm
    fill memory
mt
    memory test
pm
    put (or patch) memory
sm
    search memory

 Commands Primarily Used Within Scripts

argv
    build argv list
echo
    print string to console port
exit
    exit from the execution path of a script
gosub
    call a subroutine
goto
    branch to specified file tag
if
    conditional branching
item
    process a list of strings or items
read
    interactive shell variable entry
return
    return from a subroutine
set
    shell variable operations
sleep
    second or millisecond delay

 Ethernet Interface Commands

arp
    address resolution protocol support
dhcp
    dynamic host configuration protocol support
ether
    ethernet interface support
icmp
    internet control message protocol support

 Debug Commands

at
    set complex breakpoints
call
    call embedded function
dr
    display registers
go
    transfer control to application
pr
    put (or patch) register
prof
    configure (or dump statistics gathered from) runtime profiling through the monitor
ss
    single step
strace
    stack trace

Note that some of the commands within this set may not be ported to all targets.  Verify that the port to your target contains the commands from this set that you need.

 File Transfer Commands

tftp
    TFTP client/server support
xmodem
    XMODEM file transfer

 Miscellaneous

edit
    edit ascii TFS file
help
    display information about command
heap
    display information about the monitor heap
history
    display most recently issued commands
?
    same as help
idev
    device interface
mstat
    monitor status
mtrace
    configure and/or dump the memory trace buffer
pio
    parallel io interface
reset
    firmware generated reset
tfs
    tiny file system support
ulvl
    user-level support
version
    version of monitor build
