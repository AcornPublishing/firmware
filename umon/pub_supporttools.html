The following text discusses tools that have been written for either Sun and/or Windows95 to support various portions of the monitor and or the environment that it subscribes to.  Following is a high-level list of each of the tools currently available.

Monitor Communication and Data Transfer

Monitor Communication and Data Transfer

  • com

    • Very basic comport communication tool (runs in console mode).
  • comnet

    • Telnet-to-PC-COM-Port server.
  • moncmd

    • Interface to the monitor's UDP command interface port.
  • newmon

    • Load a new boot monitor into a target system.
  • ttftp

    • TFTP client and/or server.
  • dhcpsrvr

    • DHCP and/or BOOTP server.
  • rtcon

    • Remote target connect shell.

  File Conversion and Display

  • aout

    • Modify and/or display content of AOUT file.
  • bin2srec

    • Convert binary file to S-record format.
  • coff

    • Modify and/or display content of COFF file.
  • elf

    • Modify and/or display content of ELF file.
  • f2mem

    • Convert a series of files (binary & TFS) into a memory image for transfer to a flash device
  • fcmp

    • File comparison utility.
  • monsym

    • Convert file of tabulated data to symbol table format useable by MicroMonitor.

      Source Code Maintenance

    • cscope

      • Powerful source code browser.
      This is a modified version of the cscope obtained from SourceForge.
    • ctags

      • Source code tag generator.
      This is a modified version of Exuberant Ctags (also from SourceForge).
    • defdate

      • Generate date/time strings suitable for inclusion in a header file.

     Miscellaneous

    • maccrypt

      • Tool used to create a "backdoor" password derived from a MAC address.
    • minigzip

      • Subset of "gzip" built from the public domain zlib code for use with the monitor's unzip command.
    • title

      • Place text on the title bar of a window.
    • whence

      • Similar to "whence" in KSH, but for PC.
