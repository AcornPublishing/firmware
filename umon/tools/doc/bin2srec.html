NAME:

bin2srec

SYNOPSIS:

Convert a binary file to an S3 record file.

USAGE:

bin2srec [-b:o:] {input_file}

DESCRIPTION:

This tool takes a binary file and converts it to S3-record format.  The incoming file can be converted to exist at some address other than zero by specifying a base address; also, an offset into the input-file can be specified to "cut" int some binary.

OPTIONS:

  • -b {base address}

    • override default base address of 0;
  • -o {offset}

    • specify an offset (default is none) into the input_file at which point data conversion is started;

EXAMPLES:

  • bin2srec file

    • Convert the entire content of 'file' to S-Record format.

EXIT STATUS:

  • 0 if successful
  • 1 error
