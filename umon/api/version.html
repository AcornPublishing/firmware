API FUNCTION:

mon_version()

SYNOPSIS:

A mechanism by which the application can retrieve the monitor's build date and time stringr.

PROTOTYPE:

char *    mon_version(void);

DESCRIPTION:

This command simply returns a pointer to the same string that is displayed through the monitor's "version" command.
The format of the string is "BUILDDATE @ BUILDTIME" where...
BUILDDATE = YYYY/MM/DD
and
BUILDTIME = HH:MM:SS
