API FUNCTION:

mon_crc16()

SYNOPSIS:

CRC16 calculation.

PROTOTYPE:

unsigned short    mon_crc16(char *buffer, long nbytes);

DESCRIPTION:

Perform a crc16 calculation across the specified buffer.

Parameters:

  • char *buffer

    • data space over which the crc16 calculation is to be performed.
  • long nbytes

    • size of the buffer..
Return:
    crc.
