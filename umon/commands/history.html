NAME:

history

SYNOPSIS:

Display history of most recent commands issued at monitor command line using either a KSH/VI-like interface or the escape sequences generated by a VT100 terminal emulator for the arrow and del keys.

USAGE:

history

DESCRIPTION:

The monitor supports command line editing capabilities.  Along with that, a history is maintained of some number of the most recently executed commands.  The command line editor/history facility supports a subset of "vi-like" command sequences used by KSH:
 
esc [k-] step back through command history
esc [j+] step forward through command history
h move left one character
l  move right one character
0 move to beginning of line
$ move to end of line
x delete character
i begin insert mode at current position
a begin append mode at current position
R begin replace mode at current position
r replace single character at current position
D delete from current position to end of line
dw delete from current position to first whitespace
cw replace word from current position to first whitespace
    Alternatively (depending on whether the monitor was built with lineedit.c or ledit_vt.c), the arrow and del keys can be used if VT100 terminal emulation is active...
     
     
    up arrow step back through command history
    down arrow step forward through command history
    left arrow move left one character
    right arrow move right one character
    DEL key delete the character the cursor is on top of
