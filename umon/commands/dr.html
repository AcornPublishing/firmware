NAME:

dr

SYNOPSIS:

Display CPU registers.

USAGE:

dr [register_name]

DESCRIPTION:

Display the current set of registers used by the application.

EXAMPLES:

  • dr

    • Displays all CPU registers.
  • dr reg

    • Display CPU register reg.

    NOTES:

  • This command is only applicable after an application has been started by the go command and a breakpoint or exception has occurred.
