NAME:

mt

SYNOPSIS:

Run memory diagnostics

USAGE:

mt -[qv] {address} {size}

DESCRIPTION:

Runs walking ones and address-on-address test across the memory range specified.

OPTIONS:

  • -q

    • quit at error;
  • -v

    • verbose mode;
