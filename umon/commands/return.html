NAME:

return

SYNOPSIS:

Return from subroutine.

USAGE:

return

DESCRIPTION:

Return to the point at which the currently active subrouteinw was called via gosub..

EXAMPLES:

Within a script file:
# TOPofScript
set CNT 0
# NEXT:
gosub ECHO_CNT
if CNT lt 10 goto NEXT
exit

# ECHO_CNT:
echo $CNT
set -i CNT
sleep 1
return
